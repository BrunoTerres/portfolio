Welcome to Hero

A Django-based adaptation of the Be The Hero application, taught at the 11th edition of the week OmniStack Developed by Rocketseat. Originally done entirely with the JavaScript libraries React and React Native in conjunction with Node.js and an SQLite database. At this version I use a Python Stack, with the Django framework as a backend together with Gunicorn and Nginx to serve HTML templates and statistical files, as well as a PostgreSQL database. Be The Hero (django version) Application to connect NGOs and other institutions to people who are available to help. Powered By Django, developed by Bruno Terres!
